This tool runs a SWMM inp file using a SWMM executable.

Optionally provide a SWMM GeoPackage (GPKG) input file to generate the inp first, then run SWMM.

Provide the path to runswmm.exe, or leave it blank if the executable is in a standard path like C:\Program Files\EPA SWMM 5.2.2. Typically the default path search should be enough. If multiple SWMM installations are present the latest version is picked by default.

If report/output files are not specified, they are written next to the inp file with .rpt and .out extensions.

After the run completes, the report is scanned and up to 10 lines containing ERROR or WARNING are printed in the QGIS log. If more exist the log is truncated.
