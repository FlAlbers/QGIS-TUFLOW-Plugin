This tool runs a SWMM inp file using a SWMM executable.

Provide a SWMM inp file or a SWMM GeoPackage (GPKG). If a GPKG is provided, the inp is generated first, then SWMM runs.

Provide the path to runswmm.exe, or leave it blank if the executable is in a standard path like C:\Program Files\EPA SWMM 5.2.2. Typically the default path search should be enough. If multiple SWMM installations are present the latest version is picked by default.

If a "Save with new name" value is specified, it is used for the .inp, .rpt, and .out files. If not specified they are saved with the same name as Input.

Optional start/end date/time fields update the [OPTIONS] section of the inp before running using MM/DD/YYYY and HH:MM:SS.

After the run completes, the report is scanned and up to 10 lines containing ERROR or WARNING are printed in the QGIS log. If more exist the log is truncated.

You can click "Run in background" in the Processing dialog to keep QGIS responsive while SWMM is simulating.
